**Disciplina de Projeto e Implementação de Orientação a Objetos.**

Objetivos O aluno deverá ser capaz de desenvolver um sistema que realize as operações CRUD e controle sessão de usuário para Web seguindo os padrões MVC II, Front Controller e Command.

**Aplicativo desenvolvido**

Sistema de Financiamentos de Veículos - SIFIVEI. Um usuário logado poderá realizar ações de Incluir, Alterar, Pesquisar e Excluir dados de clientes, veículos e financiamentos. 

**Técnologias Usadas**

•	Java
•	Servlet
•	jsp
•	Javascript
•	Postgres SQL
 
**Padrões usados**

•	MVC
•	Front Controller
•	Command
•	DAO

**Frameworks usados**
•	Jquery
•	Bootstrap
•	Hibernate
